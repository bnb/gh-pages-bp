Code Highlighting

Code highlighting is obviously important for gh-pages sites because we need to display code as examples of how to (or not to) do specific things with out projects.

gh-pages-bp comes with Prism for code highlighting. By default, gh-pages=bp comes with the standard theme, as well as syntax highlighting for markup, CSS (plus CSS Extras), JavaScript, PHP (plus PHP Extras), CoffeeScript, SASS, SQL, and Ruby. If the language you want to use isn't on this list, check out the Prism.js download page where you can customize Prism to fit your needs.

Here's an example code block (it's from the Node.js documentation, with JavaScript highlighting): 

console.log('a starting');
exports.done = false;
var b = require('./b.js');
console.log('in a, b.done = %j', b.done);
exports.done = true;
console.log('a done');

gh-pages-bp

A simple, pretty gh-pages boilerplate.

About Docs GitHub

gh-pages-bp offers

Well-designed

gh-pages-bp is well designed from the beginning, so there's no need to fret over what people think. It allows you to just plug in your info and save.

Easy CLI (Soon!)

The CLI is a WIP. It will allow you to deploy gh-pages-bp easily and efficiently. Commands are designed to make things easy. No fuss, no muss.

Open-Source Software

gh-pages-bp is an OSS project, and it's meant to give something back to the community that has already given so much.

BassCSS

Powerful CSS

BassCSS is the CSS framework behind the lovely design of gh-pages-bp. It's extremely powerful, but also simple - not bloated.

Flexbox

Flexbox, flexbox, flexbox. Flexbox is an amazing new CSS property that allows extreme control over the positioning of elements. It's a must have.

Color Combinations

BassCSS uses mrmrs' colors as default colors for styling. This extension is extremely useful, and gives real creedence to using regular color names, albiet in the Markup.

Optional Modules

BassCSS has optional modules that allow you to customize your gh-pages site even more. You should definitely look into these if you want to add more functionality to gh-pages-bp.
